# Collection of Agent Skills

## Created here

- `coding-best-practices` — General coding principles; references `web-ui-best-practices` and `react-best-practices` for frontend/React work.
- `git-best-practices` — Safe-by-default git workflow and Conventional Commits.
- `web-ui-best-practices` — Opinionated UI constraints (inspired by others; documented in the folder).
